# 界面截图

本目录存放系统各功能模块的界面截图，用于 `使用指南.md` 文档。

## ⚠️ 重要提示

**当前图片文件尚未添加。** 请按照以下步骤添加截图：

1. 启动系统并访问各个功能页面
2. 使用浏览器截图工具或系统截图工具截取界面
3. 将截图保存为 PNG 格式，文件名如下：
   - `homepage.png` - 首页截图
   - `candidates.png` - 候选人管理截图
   - `automation.png` - 自动化工作流截图
   - `jobs.png` - 岗位画像截图
4. 将图片文件放入本目录 (`docs/images/`)

## 图片文件列表

- `homepage.png` - 首页/仪表板界面截图
- `candidates.png` - 候选人管理界面截图（包含候选人列表和详情视图）
- `automation.png` - 自动化工作流界面截图
- `jobs.png` - 岗位画像界面截图

## 截图说明

### 首页界面 (homepage.png)
- 显示欢迎横幅
- 四个统计卡片（已筛选候选人总数、新消息、新招呼、运行中工作流）
- 四个功能入口卡片（手动招聘、自动化工作流、岗位画像、API 文档）
- 更新日志区域

### 候选人管理界面 (candidates.png)
- 左侧：候选人列表，包含筛选条件和"全部分析"按钮
- 右侧：候选人详情，包含简历标签、AI 分析结果、对话历史等
- 顶部：标签页切换（推荐牛人、新招呼、沟通中、牛人已读未回）

### 自动化工作流界面 (automation.png)
- 远程访问 URL 配置区域
- 工作流配置选项（推荐牛人、新招呼、沟通中、追结果）
- 岗位选择和阈值设置
- 控制按钮（启动、暂停、下一步、停止、清空日志）
- 实时事件日志区域

### 岗位画像界面 (jobs.png)
- 岗位标签页切换
- 基本信息表单（岗位 ID、岗位名称）
- 岗位详情（公司背景、主要职责、任职要求、岗位概述、理想人选）
- 追问问题列表
- 候选人筛选条件（JSON 格式）
- 关键词管理（正向关键词、负向关键词）

## 如何添加新截图

1. 使用浏览器截图工具或系统截图工具截取界面
2. 保存为 PNG 格式，文件名使用英文小写字母和下划线
3. 将图片文件放入本目录
4. 在 `使用指南.md` 中添加图片引用：`![描述](docs/images/文件名.png)`
